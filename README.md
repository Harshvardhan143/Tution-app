# EduSpark Academy

A modern Tuition Management System built with React Native (Expo) and Next.js.

## 🏗️ Architecture
This is a monorepo containing:
- `backend/` — Next.js 16 API & Admin Web Panel
- `mobile/` — React Native (Expo SDK 52) App for Students and Staff

## 🌿 Git Branching Strategy
We use a GitFlow-inspired strategy:
- `main`: Production-ready code
- `develop`: Integration branch
- `feature/*`: Feature development (Create PR to merge into `develop`)
